The default REST base URL is `https://api.linxio.com/api`. Authenticated
requests use `Authorization: Bearer <token>`.

## Response Envelope

Most list endpoints return a pagination envelope:

```json title="Paginated response" lineNumbers
{
  "page": 1,
  "limit": 100,
  "total": 123,
  "data": []
}
```

<ReferenceGrid>
  <div>
    <ReferenceHeader
      title="List vehicles"
      method="GET"
      path="/vehicles/fields/json"
      sdk="linxio.vehicles.list(params)"
    >
      Returns a paginated list of vehicles. Use `fields[]` to keep responses
      small in scripts and scheduled jobs.
    </ReferenceHeader>

    <FieldTable
      title="Query parameters"
      fields={[
        {
          name: "limit",
          type: "number",
          description: "Number of records per page.",
        },
        {
          name: "page",
          type: "number",
          description: "Page number to read.",
          defaultValue: "1",
        },
        {
          name: "fields[]",
          type: "string[]",
          description: "Vehicle fields to include in each row, such as id, regNo, status, or lastLoggedAt.",
        },
        {
          name: "sort",
          type: "string",
          description: "Sort expression accepted by Linxio for this endpoint.",
        },
      ]}
    />
  </div>

  <CodeRail>

```ts title="SDK" lineNumbers
const { data, error, meta } = await linxio.vehicles.list({
  fields: ["id", "regNo", "lastLoggedAt"],
  limit: 100,
  page: 1,
});

if (error) {
  throw error;
}

console.log(meta.total, data);
```

```bash title="HTTP" lineNumbers
curl "https://api.linxio.com/api/vehicles/fields/json?fields[]=id&fields[]=regNo&limit=100&page=1" \
  -H "Authorization: Bearer $LINXIO_TOKEN"
```

  </CodeRail>
</ReferenceGrid>

  <div>
    <ReferenceHeader
      title="Get vehicle routes"
      method="GET"
      path="/vehicles/{vehicleId}/routes"
      sdk="linxio.routes.getVehicleRoutes(vehicleId, params)"
    >
      Returns trip and stop history for one vehicle over a date range. Request
      coordinate fields only when you need full point history.
    </ReferenceHeader>

    <FieldTable
      title="Path and query parameters"
      fields={[
        {
          name: "vehicleId",
          type: "number | string",
          required: true,
          description: "Internal Linxio vehicle identifier.",
        },
        {
          name: "dateFrom",
          type: "string",
          required: true,
          description: "Start date/time in Linxio's accepted ISO-style format.",
        },
        {
          name: "dateTo",
          type: "string",
          required: true,
          description: "End date/time in Linxio's accepted ISO-style format.",
        },
        {
          name: "fields[]",
          type: "string[]",
          description: "Additional route fields to include. coordinates can be large.",
        },
      ]}
    />
  </div>

  <CodeRail>

```ts title="SDK" lineNumbers
const routes = await linxio.routes.getVehicleRoutes(vehicleId, {
  dateFrom: "2026-06-01T00:00:00+08:00",
  dateTo: "2026-06-08T00:00:00+08:00",
});

if (routes.error) {
  throw routes.error;
}
```

  </CodeRail>
</ReferenceGrid>

      title="Create a geofence"
      method="POST"
      path="/areas"

          required: true,
          description: "Geofence geometry type.",
        },
        {
          name: "coordinates",
          type: "{ lat: number; lng: number }[]",
          required: true,
          description: "Geometry coordinates. Circles usually use one center point.",
        },
        {
          name: "radius",
          type: "number",
          description: "Radius in metres for circle geofences.",
        },
      ]}
    />
  </div>

  <CodeRail>

```ts title="SDK" lineNumbers
const created = await linxio.geofences.create({
  name: "Temporary Test Area",
  type: "circle",
  radius: 150,
  coordinates: [{ lat: -31.95, lng: 115.86 }],
});

if (created.error) {
  throw created.error;
}
```

  </CodeRail>
