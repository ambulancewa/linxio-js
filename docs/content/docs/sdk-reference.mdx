## Create a client

<ReferenceGrid>
  <div>

        },
        {
          name: "refreshToken",
          type: "string",

        },
        {
          name: "timeoutMs",

          defaultValue: "30000",
        },
        {

          description: "Retry configuration for idempotent requests that fail with transient network or 5xx errors.",
        },
        {
          name: "fetch",

          description: "Custom fetch implementation for tests, proxies, or non-standard runtimes.",
        },
        {
          name: "realtimeBaseUrl",

          defaultValue: "https://track.linxio.com",

  <CodeRail>

```ts title="Client setup" lineNumbers
import { createClient } from "linxio-js";

const linxio = createClient({
  timeoutMs: 30_000,
  retry: {
    retries: 3,
    delayMs: 1_000,
  },
});
```

## Result Contract

Domain service methods return result objects so scripts can branch without
wrapping every call in `try/catch`.

<FieldTable
  title="LinxioResult<T>"
  fields={[
    {
      name: "data",
      type: "T | null",
      description: "Successful response payload. Null when the call failed.",
    },
    {
      name: "error",
      type: "LinxioError | null",
      description: "Typed SDK error. Null when the call succeeded.",
    },
  ]}
/>

<FieldTable
  title="LinxioPageResult<T>"
  fields={[
    {
      name: "data",
      type: "T[]",
      description: "Current page of records.",
    },
    {
      name: "error",
      type: "LinxioError | null",
      description: "Typed SDK error. Null when the page was loaded successfully.",
    },
    {
      name: "meta",
      type: "{ page: number; limit: number; total: number }",
      description: "Normalized pagination metadata.",
    },
    {
      name: "aggregations",
      type: "unknown",
      description: "Aggregation payload returned by Linxio when the endpoint includes it.",
    },
  ]}
/>

  <CodeRail>

```ts title="Auto-pagination" lineNumbers
const { data: vehicles, error } = await linxio.vehicles.iterate({
  fields: ["id", "regNo", "lastLoggedAt"],
  limit: 100,
});

if (error) {
  throw error;
}

for (const vehicle of vehicles) {
  console.log(vehicle.id, vehicle.regNo);
}
```

```ts title="Stream lazily" lineNumbers
for await (const vehicle of linxio.vehicles.stream({ limit: 100 })) {
  console.log(vehicle.id, vehicle.regNo);
}
```

## Advanced requests

<ReferenceGrid>
  <div>
    <ReferenceHeader title="Advanced requests" sdk="linxio.request(method, path, options)">
      Use raw requests for tenant-specific or newly discovered endpoints that
      are not represented by a domain service yet. Raw requests use the same
      auth, timeout, retry, and token-refresh behavior as service methods.
    </ReferenceHeader>

    <FieldTable
      title="Parameters"
      fields={[
        {
          name: "method",
          type: "GET | POST | PATCH | PUT | DELETE",
          required: true,
          description: "HTTP method to send.",
        },
        {
          name: "path",
          type: "string",
          required: true,
          description: "API path relative to the configured base URL, or an absolute URL.",
        },
        {
          name: "options.params",
          type: "QueryParams",
          description: "Query parameters to append to the request URL.",
        },
        {
          name: "options.body",
          type: "unknown",
          description: "JSON body or request body passed to fetch.",
        },
      ]}
    />
  </div>

  <CodeRail>

```ts title="Raw request" lineNumbers
const result = await linxio.request("GET", "/country/list");
console.log(result);
```

```ts title="POST with body" lineNumbers
const result = await linxio.request("POST", "/vehicles", {
  body: {
    regNo: "ABC-123",
    type: "car",
  },
});
```

  </CodeRail>
</ReferenceGrid>
