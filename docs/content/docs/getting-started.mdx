## Create a client

<ReferenceGrid>
  <div>
    <ReferenceHeader title="Create a client" sdk="createClient(options)" hideTitle={true}>
      Instantiate a client once per process. Pass optional configuration for
      timeout, retry behaviour, and a custom fetch implementation.
    </ReferenceHeader>

    <FieldTable
      fields={[
        {
          name: "timeoutMs",
          type: "number",
          description: "Per-request timeout. Throws LinxioTimeoutError when exceeded.",
          defaultValue: "30000",
        },
        {
          name: "retry",
          type: "{ retries?: number; delayMs?: number }",
          description: "Retry policy for transient failures on idempotent requests.",
        },
      ]}
    />
  </div>

  <CodeRail>

```ts title="Client setup"
import { createClient } from "linxio-js";

const linxio = createClient({
  timeoutMs: 30_000,
  retry: {
    retries: 3,
    delayMs: 1_000,
  },
});
```

  </CodeRail>
</ReferenceGrid>

## Authenticate

<ReferenceGrid>
  <div>
    <ReferenceHeader title="Authenticate" method="POST" path="/login" sdk="linxio.auth.login(payload)">
      Login stores the session in memory. Subsequent service calls include the
      bearer token automatically. Store the session yourself if your process
      needs to survive restarts.
    </ReferenceHeader>
  </div>

  <CodeRail>

```ts title="Login"
const login = await linxio.auth.login({
  email: process.env.LINXIO_EMAIL!,
  password: process.env.LINXIO_PASSWORD!,
});

if (login.error) {
  throw login.error;
}

// Persist tokens across restarts
const session = linxio.session();
await saveSession(session);

// Restore on next startup
const linxio2 = createClient(await loadSession());
```

  </CodeRail>
</ReferenceGrid>

## Read vehicles

<ReferenceGrid>
  <div>
    <ReferenceHeader title="Read vehicles" sdk="linxio.vehicles.list(params)">
      Paginated list with field projection. Use `iterate()` to auto-page
      everything into one array, or `stream()` for lazy async iteration.
    </ReferenceHeader>
  </div>

  <CodeRail>

```ts title="Single page"
const { data: vehicles, error, meta } = await linxio.vehicles.list({
  fields: ["id", "regNo", "lastLoggedAt"],
  limit: 100,
  page: 1,
  sort: "regNo",
});

if (error) throw error;
console.log(meta.total, vehicles);
```

```ts title="All pages at once"
const { data: all, error } = await linxio.vehicles.iterate({
  fields: ["id", "regNo", "lastLoggedAt"],
  limit: 100,
});

if (error) throw error;
for (const v of all) console.log(v.id, v.regNo);
```

```ts title="Lazy stream"
for await (const vehicle of linxio.vehicles.stream({ limit: 100 })) {
  console.log(vehicle.id, vehicle.regNo);
}
```

  </CodeRail>
</ReferenceGrid>
