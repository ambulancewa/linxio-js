---
title: Fuel
description: Fuel transaction records, summaries, vehicle fuel rows, cards, fuel types, and transaction assignment.
---

<ImportantNote>
Fuel endpoints are dashboard-derived and can include tenant-specific columns.
The SDK keeps the common fields typed and leaves additional columns available on
the returned record.
</ImportantNote>

## fuel.records()

<ReferenceGrid>
  <div>
    <ReferenceHeader title="List fuel transactions" method="GET" path="/fuel-cards/json" sdk="linxio.fuel.records(params)">
      Loads one page of fuel transaction records.
    </ReferenceHeader>

    <FieldTable
      title="Input"
      fields={[
        {

          type: "number",

          type: "number",
          description: "Page number to request.",
          defaultValue: "1",
        },
        {
          name: "params.fields",
          type: "FuelRecordField[]",
          description: "Optional fields such as transactionDate, vehicleIds, driverId, refueled, total, fuelPrice, or petrolStation.",
        },
        {
          name: "params.[filter]",
          type: "QueryValue",
          description: "Additional fuel report filters forwarded as query parameters.",
        },
      ]}
    />

    <FieldTable
      title="Returns"
      fields={[
        {

          description: "Fuel transaction records for the requested page.",

        {
          name: "meta",
          type: "LinxioPaginationMeta | null",
          description: "Pagination metadata.",
        },
        {

        },
      ]}
    />
