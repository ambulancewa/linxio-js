---
title: Raw HTTP
description: Low-level request helpers, HTTP client options, retry policy, and response parsing.
---

Use raw HTTP when you need a tenant-specific or newly discovered endpoint that
is not represented by a domain service yet. Raw requests use the same auth,
timeout, retry, and token-refresh behavior as service methods.

## client.request()

<ReferenceGrid>
  <div>
    <ReferenceHeader title="Send a raw API request" sdk="linxio.request(method, path, options)">
      Sends a raw request through the client's HTTP layer.
    </ReferenceHeader>

    <CautionNote>Raw HTTP methods are not intended for production use. They are not typed, do not have any guardrails, and are provided for advanced use cases and testing. Where possible, use the domain services instead.</CautionNote>

    <FieldTable
      title="Input"
      fields={[
        {
          name: "method",
          type: "GET | POST | PATCH | PUT | DELETE | HEAD | OPTIONS",
          required: true,
          description: "HTTP method to send.",
        },
        {
          name: "path",
          type: "string",
          required: true,
          description: "API path relative to the configured base URL, or an absolute URL.",
        },
        {
          name: "options.params",
          type: "QueryParams",
          description: "Query parameters appended to the URL.",
        },
        {
          name: "options.body",
          type: "unknown",
          description: "JSON body, FormData, URLSearchParams, string, ArrayBuffer, or Blob.",
        },
        {
          name: "options.responseType",
          type: "json | text | blob | arrayBuffer | raw",
          description: "Response parser override.",
          defaultValue: "json",
        },
      ]}
    />

    <FieldTable
      title="Headers"
      fields={[
        { name: "options.headers", type: "HeadersInit", description: "Additional HTTP headers for custom or newly discovered endpoints." },
        { name: "options.idempotencyKey", type: "string", description: "Sets the Idempotency-Key header for caller-managed retry safety." },
      ]}
    />

    <FieldTable
      title="Returns"
      fields={[
        { name: "data", type: "Promise<TResponse>", description: "Parsed response. Raw request methods throw typed SDK errors instead of returning LinxioResult." },
        { name: "throws", type: "LinxioApiError | LinxioNetworkError | LinxioTimeoutError", description: "Typed SDK error when the request fails." },
      ]}
    />
  </div>

  <CodeRail>

```ts title="Raw GET" lineNumbers
const countries = await linxio.request("GET", "/country/list");
console.log(countries);
```

```ts title="Raw POST" lineNumbers
const vehicle = await linxio.request("POST", "/vehicles", {
  body: {
    regNo: "TEST-001",
    defaultLabel: "Temporary SDK Test Vehicle",
  },
});
```

  </CodeRail>
</ReferenceGrid>
