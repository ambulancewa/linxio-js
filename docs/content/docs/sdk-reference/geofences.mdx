---
title: Geofences
description: Area and area-group lifecycle methods, including create, update, archive, restore, and delete.
---

<Note>
Linxio calls geofences `areas` in the REST API. 

This SDK uses `linxio.geofences` to make that domain clearer.
</Note>

## geofences.list()

<ReferenceGrid>
  <div>
    <ReferenceHeader title="List geofences" method="GET" path="/areas" sdk="linxio.geofences.list(params)">
      Loads one page of geofence areas.
    </ReferenceHeader>

    <ImportantNote>If your tenant has many complicated geofences, or many geofence areas, the API can return extremely large response payloads. Consider whether using `geofences.list()` in production implementations is appropriate for your use case.</ImportantNote>

    <FieldTable
      title="Input"
      fields={[
        {
          name: "params.limit",
          type: "number",
          description: "Number of geofences to request.",
        },
        {
          name: "params.page",
          type: "number",
          description: "Page number to request.",
          defaultValue: "1",
        },
        {
          name: "params.sort",
          type: "string",
          description: "Sort expression accepted by Linxio.",
        },
        {
          name: "params.fields",
          type: "string[]",
          description: "Optional field projection.",
        },
      ]}
    />

    <FieldTable
      title="Returns"
      fields={[
        {
          name: "data",
          type: "LinxioGeofence[] | null",
          description: "Geofence records for the requested page.",
        },
        {
          name: "meta",
          type: "LinxioPaginationMeta | null",
          description: "Pagination metadata.",
        },
        {
          name: "error",
          type: "LinxioError | null",
          description: "Typed SDK error when the page fails.",
        },
      ]}
    />
  </div>

  <CodeRail>

```ts title="List geofences" lineNumbers
const areas = await linxio.geofences.list({
  limit: 100,
  page: 1,
});

if (areas.error) {
  throw areas.error;
}
```

  </CodeRail>
</ReferenceGrid>

## geofences.iterate()

<ReferenceGrid>
  <div>
    <ReferenceHeader title="Load all geofences" method="GET" path="/areas" sdk="linxio.geofences.iterate(params)">
      Auto-paginates geofence pages into one result object.

    />

    <FieldTable
      title="Returns"

        {
          name: "data",

          description: "Flat array containing every loaded geofence.",
        },
        {
          name: "error",
          type: "LinxioError | null",
          description: "Typed SDK error if any page fails.",
        },
      ]}
    />
  </div>
