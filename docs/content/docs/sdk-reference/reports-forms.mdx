---
title: Reports and Digital Forms
description: Scheduled report lifecycle methods and digital form answer retrieval.
---

Reports and digital form endpoints are dashboard-derived. Use the typed payloads
for common fields and preserve any tenant-specific fields returned by Linxio.

## reports.scheduled()

<ReferenceGrid>
  <div>
    <ReferenceHeader title="List scheduled reports" method="GET" path="/scheduled-report" sdk="linxio.reports.scheduled(params)">

## digitalForms.answerPdf()

<ReferenceGrid>
  <div>
    <ReferenceHeader title="Download a form answer PDF" method="GET" path="/digital-form/answer/{answerId}/pdf" sdk="linxio.digitalForms.answerPdf(answerId)">
      Downloads a digital form answer as a PDF Blob.
    </ReferenceHeader>

    <FieldTable
      title="Input"
      fields={[
        {
          name: "answerId",
          type: "LinxioId",
          required: true,
          description: "Digital form answer identifier.",
        },
      ]}
    />

    <FieldTable
      title="Returns"
      fields={[
        {
          name: "data",
          type: "Blob | null",
          description: "PDF binary data.",
        },
        {
          name: "error",
          type: "LinxioError | null",
          description: "Typed SDK error when the PDF cannot be downloaded.",
        },
      ]}
    />
  </div>

  <CodeRail>

```ts title="Download answer PDF" lineNumbers
import { writeFile } from "node:fs/promises";

const pdf = await linxio.digitalForms.answerPdf(answerId);

if (pdf.error) {
  throw pdf.error;
}

const bytes = new Uint8Array(await pdf.data.arrayBuffer());
await writeFile("answer.pdf", bytes);
```

  </CodeRail>
</ReferenceGrid>
