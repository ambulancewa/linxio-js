---
title: Authentication
description: Login, OTP, token refresh, current user, logout, and session hydration methods.
---

Use the auth service to create and maintain a Linxio session. Successful
`login`, `verifyOtp`, and `refresh` calls update the client instance in memory,
so later service calls send the bearer token automatically.

## auth.login()

<ReferenceGrid>
  <div>
    <ReferenceHeader title="Log in to Linxio" method="POST" path="/login" sdk="linxio.auth.login(request)">
      Authenticates a Linxio user and stores the returned session on the client.
    </ReferenceHeader>

    <FieldTable
      title="Input"
      fields={[

        },
        {
          name: "request.password",
          type: "string",
          required: true,
          description: "Password for the user. Load this from a secret store or environment variable.",
        },
        {
          name: "request.domain",
          type: "string",
          description: "Optional tenant or hosted-domain hint. Most API integrations can omit it.",
        },
      ]}
    />

    <FieldTable
      title="Returns"
      fields={[
        {
          name: "data.token",
          type: "string",
          description: "JWT bearer token used for authenticated API requests.",
        },
        {
          name: "data.refreshToken",
          type: "string",

        },
        {

        },
        {

          description: "True when the account must complete OTP before the session is usable.",

        },

```ts title="Login" lineNumbers
const login = await linxio.auth.login({
  email: process.env.LINXIO_EMAIL!,
  password: process.env.LINXIO_PASSWORD!,

  throw login.error;
}

if (login.data.otp_required) {
  console.log("Complete OTP before using the session.");
}
```

## auth.refresh()

<ReferenceGrid>
  <div>
    <ReferenceHeader title="Refresh JWT auth token" method="POST" path="/token/refresh" sdk="linxio.auth.refresh(refreshToken)">
      Refreshes a JWT manually and updates the client session. Most callers do
      not need this because the HTTP layer refreshes automatically after a
      recoverable 401.
    </ReferenceHeader>

    <FieldTable
      title="Input"
      fields={[
        {
          name: "refreshToken",
          type: "string",
          required: true,
          description: "Refresh token returned by login or a previous refresh call.",
        },
      ]}
    />

    <FieldTable
      title="Returns"
      fields={[
        {
          name: "data",
          type: "LinxioSession",
          description: "Updated in-memory token state containing token, refreshToken, and expireAt when available.",
        },
        {
          name: "error",
          type: "LinxioError | null",
          description: "Typed SDK error when refresh fails.",
        },
      ]}
    />
  </div>

  <CodeRail>

```ts title="Manual refresh" lineNumbers
const session = linxio.session();

if (session.refreshToken) {
  const refreshed = await linxio.auth.refresh(session.refreshToken);

  if (refreshed.error) {
    throw refreshed.error;
  }
}
```

  </CodeRail>
</ReferenceGrid>

## auth.me()

<ReferenceGrid>
  <div>
    <ReferenceHeader title="Get current user" method="GET" path="/me" sdk="linxio.auth.me(fields)">
      Fetches the current authenticated user. Pass fields when you only need a
      small subset of the user profile.
    </ReferenceHeader>

    <FieldTable
      title="Input"
      fields={[
        {
          name: "fields",
          type: "string[]",
          description: "Optional field projection mapped to fields[] query parameters.",
        },
      ]}
    />

    <FieldTable
      title="Returns"
      fields={[
        {
          name: "data",
          type: "LinxioCurrentUser",
          description: "Current user profile, permissions, team information, and tenant-specific fields returned by Linxio.",
        },
        {
          name: "error",
          type: "LinxioError | null",
          description: "Typed SDK error when the user cannot be loaded.",
        },
      ]}
    />
  </div>

  <CodeRail>

```ts title="Current user" lineNumbers
const me = await linxio.auth.me(["id", "email", "teamType", "team"]);

if (me.error) {
  throw me.error;
}

console.log(me.data.email, me.data.teamType);
```

  </CodeRail>
</ReferenceGrid>
