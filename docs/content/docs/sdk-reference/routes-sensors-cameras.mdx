## routes.getVehicleRoutes()

<ReferenceGrid>
  <div>
    <ReferenceHeader title="Get vehicle route history" method="GET" path="/vehicles/{vehicleId}/routes" sdk="linxio.routes.getVehicleRoutes(vehicleId, params)">
      Fetches route history for one vehicle. Request `coordinates` only when
      you need every recorded point.
    </ReferenceHeader>
    
    <ImportantNote>Vehicle route history can return large payloads. Use date ranges and field projection deliberately. If your tenant has many vehicles and/or vehicles with long route histories, <strong>avoid using `routes.getVehicleRoutes()` in production implementations.</strong></ImportantNote>

    <FieldTable
      title="Input"
      fields={[
        {
          name: "vehicleId",
          type: "LinxioId",
          required: true,
          description: "Vehicle identifier.",
        },
        {
          name: "params.dateFrom",
          type: "ISODateString",
          description: "Start date/time.",
        },
        {
          name: "params.dateTo",
          type: "ISODateString",
          description: "End date/time.",
        },
        {
          name: "params.fields",
          type: "RouteField[]",
          description: "Route fields to include in the response projection.",
        },
        {
          name: "params.limit",
          type: "number",
          description: "Limit accepted by the route endpoint.",
        },
        {
          name: "params.page",
          type: "number",
          description: "Page accepted by the route endpoint.",
        },
      ]}
    />

    <FieldTable
      title="Returns"
      fields={[
        {
          name: "data",
          type: "LinxioVehicleRoutesGroup[] | null",
          description: "Route groups by vehicle/driver, each with route segments.",
        },
        {
          name: "error",
          type: "LinxioError | null",
          description: "Typed SDK error when routes cannot be loaded.",
        },
      ]}
    />
  </div>

  <CodeRail>

```ts title="Route history" lineNumbers
const routes = await linxio.routes.getVehicleRoutes(vehicleId, {
  dateFrom: "2026-06-01T00:00:00+08:00",
  dateTo: "2026-06-08T00:00:00+08:00",
  fields: ["address"],
});

if (routes.error) {
  throw routes.error;
}
```

```ts title="Include coordinates only when needed" lineNumbers
const routesWithPoints = await linxio.routes.getVehicleRoutes(vehicleId, {
  dateFrom,
  dateTo,
  fields: ["coordinates"],
});
```

  </CodeRail>
</ReferenceGrid>

## sensors.iterateDeviceTemperatureHumidityReport()

<ReferenceGrid>
  <div>
    <ReferenceHeader title="Load all device temperature and humidity readings" method="GET" path="/devices/sensors/report/temp-and-humidity" sdk="linxio.sensors.iterateDeviceTemperatureHumidityReport(params)">
      Auto-paginates the device temperature/humidity report into one result
      object.
    </ReferenceHeader>
    
    <ImportantNote>Device temperature and humidity reports can return large payloads, especially when using the iterate method. Use pagination, date ranges, and field projection deliberately. Avoid using `iterate()` in production implementations. It is better to use <a href="/docs/sdk-reference/routes-sensors-cameras#sensorsstreamdevicetemperaturehumidityreport" className="text-fd-foreground underline decoration-fd-muted-foreground/35 underline-offset-4 transition-colors hover:text-fd-primary hover:decoration-fd-primary/70">`sensors.streamDeviceTemperatureHumidityReport()`</a> for lazy async iteration where possible.</ImportantNote>

    <FieldTable
      title="Input"
      fields={[
        {
          name: "params",
          type: "LinxioSensorReportParams",
          description: "Same parameters as deviceTemperatureHumidityReport().",
        },
      ]}
    />

    <FieldTable
      title="Returns"
      fields={[
        {
          name: "data",
          type: "LinxioTemperatureHumidityReading[] | null",
          description: "Flat array containing every loaded reading.",
        },
        {
          name: "error",
          type: "LinxioError | null",
          description: "Typed SDK error if any page fails.",
        },
      ]}
    />
  </div>

  <CodeRail>

```ts title="Iterate sensor report" lineNumbers
  await linxio.sensors.iterateDeviceTemperatureHumidityReport({

    endDate,
    limit: 100,
  });

if (error) {
  throw error;
}
```

  </CodeRail>
