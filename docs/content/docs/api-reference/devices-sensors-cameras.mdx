---
title: Devices, Sensors, and Cameras
description: Device inventory, install/uninstall, coordinate history, sensor reports, and camera event endpoints.
---

Device endpoints manage hardware inventory and related telemetry. Sensor and
camera endpoints are a mix of publicly documented and dashboard-derived paths.

<ReferenceGrid>
  <div>
    <ReferenceHeader title="List devices" method="GET" path="/devices/json" sdk="linxio.devices.list(params)">
