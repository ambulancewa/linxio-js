---
title: Fuel
description: Fuel transaction, summary, card, fuel type, and transaction assignment endpoints.
---

Fuel endpoints are dashboard-derived and can include tenant-specific fields.

## GET /fuel-cards/json

<ReferenceGrid>
  <div>
    <ReferenceHeader title="List fuel transactions" method="GET" path="/fuel-cards/json" sdk="linxio.fuel.records(params)">
      Returns a paginated list of fuel transactions. The SDK also exposes this
      path as `fuel.cards()` for fuel-card-shaped reads.
    </ReferenceHeader>

    <FieldTable title="Query parameters" fields={[
      { name: "limit", type: "number", description: "Number of records per page." },
      { name: "page", type: "number", description: "Page number to read." },
      { name: "fields[]", type: "string[]", description: "Optional fields such as transactionDate, vehicleIds, refueled, total, fuelPrice, or petrolStation." },
      { name: "[filter]", type: "QueryValue", description: "Fuel report filters accepted by Linxio." },
    ]} />
    <FieldTable title="Response fields" fields={[
      { name: "data", type: "LinxioFuelRecord[]", description: "Fuel transaction records for the current page." },
      { name: "page / limit / total", type: "number", description: "Pagination metadata." },
    ]} />
  </div>

  <CodeRail>

```bash title="Request" lineNumbers
curl "https://api.linxio.com/api/fuel-cards/json?fields[]=transactionDate&fields[]=total&limit=100" \
  -H "Authorization: Bearer $LINXIO_TOKEN"
```

  </CodeRail>
</ReferenceGrid>

## PATCH `/fuel-cards/record/{recordId}`

<ReferenceGrid>
  <div>
    <ReferenceHeader title="Assign fuel transaction to vehicle" method="PATCH" path="/fuel-cards/record/{recordId}" sdk="linxio.fuel.assignTransaction(recordId, vehicleId)">
      Assigns a fuel transaction record to a vehicle.
    </ReferenceHeader>

    <FieldTable title="Path and body parameters" fields={[
      { name: "recordId", type: "LinxioId", required: true, description: "Fuel transaction record identifier." },
      { name: "vehicleId", type: "LinxioId", required: true, description: "Vehicle identifier sent in the JSON body." },
    ]} />
    <FieldTable title="Response fields" fields={[{ name: "id", type: "LinxioId", description: "Updated fuel transaction identifier." }]} />
  </div>

  <CodeRail>

```bash title="Request" lineNumbers
curl -X PATCH https://api.linxio.com/api/fuel-cards/record/9001 \
  -H "Authorization: Bearer $LINXIO_TOKEN" \
  -H "Content-Type: application/json" \
  --data '{
    "vehicleId": 304
  }'
```

  </CodeRail>
</ReferenceGrid>
