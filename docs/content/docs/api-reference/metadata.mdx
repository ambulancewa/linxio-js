---
title: Metadata
description: Read-only reference data and tenant settings endpoints.
---

Metadata endpoints return dropdown options, tenant settings, and feature flags.
Most shapes are tenant-configurable, so the SDK preserves unknown fields.

<ReferenceGrid>
  <div>
    <ReferenceHeader title="List countries" method="GET" path="/country/list" sdk="linxio.metadata.countries()">

    </ReferenceHeader>
