The default REST base URL is `https://api.linxio.com/api`. Authenticated
requests require an `Authorization: Bearer <token>` header from the login
endpoint.

The API reference is split by endpoint family. Publicly documented endpoints
are included together with dashboard-derived endpoints represented in the SDK.

## Response envelope

Most list endpoints return a pagination envelope:

```json title="Paginated response" lineNumbers
{
  "page": 1,
  "limit": 100,
  "total": 123,
  "data": [],
  "aggregations": null
}
```

Most entity endpoints return the entity directly. Endpoints returning `204 No
Content` are represented by `LinxioResult<void>` in the SDK.
