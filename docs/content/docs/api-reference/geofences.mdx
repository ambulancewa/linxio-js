---
title: Geofences
description: Geofence area and area-group HTTP endpoints.
---

The REST API names geofences as `areas`. Use archive endpoints for reversible
operations and delete endpoints only when you intend to remove records
permanently.

## GET /areas

<ReferenceGrid>
  <div>
    <ReferenceHeader title="List geofences" method="GET" path="/areas" sdk="linxio.geofences.list(params)">
      Returns a paginated list of geofence areas.
    </ReferenceHeader>

    <RequiredHeaders />

    <FieldTable title="Request query parameters" iconName="forward"  fields={[
      { name: "limit", type: "number", description: "Number of records per page." },
      { name: "page", type: "number", description: "Page number to read." },
      { name: "fields[]", type: "string[]", description: "Optional field projection." },
      { name: "sort", type: "string", description: "Sort expression accepted by Linxio." },
    ]} />
    <FieldTable title="Response body fields" iconName="bodyParams" fields={[
      { name: "data", type: "LinxioGeofence[]", description: "Geofence records for the current page." },
      { name: "page / limit / total", type: "number", description: "Pagination metadata." },
    ]} />
  </div>

  <CodeRail>

```bash title="Request" lineNumbers
curl "https://api.linxio.com/api/areas?limit=100&page=1" \
  -H "Authorization: Bearer $LINXIO_TOKEN"
```

  </CodeRail>
</ReferenceGrid>

## POST /areas

<ReferenceGrid>
  <div>
    <ReferenceHeader title="Create a geofence" method="POST" path="/areas" sdk="linxio.geofences.create(payload)">
      Creates a geofence area.
    </ReferenceHeader>

    <RequiredHeaders json />

    <FieldTable title="Request body parameters" iconName="urlParams" fields={[
      { name: "name", type: "string", required: true, description: "Display name." },
      { name: "type", type: "GeofenceType", required: true, description: "Geometry type." },
      { name: "coordinates", type: "LatLng[]", description: "Geometry coordinates. Circles typically use one centre point." },
      { name: "radius", type: "number", description: "Circle radius in metres." },
      { name: "color", type: "string", description: "Display colour accepted by Linxio." },
    ]} />
    <FieldTable title="Response body fields" iconName="bodyParams" fields={[{ name: "id", type: "LinxioId", description: "Created geofence area ID." }]} />
  </div>

  <CodeRail>

```bash title="Request" lineNumbers
curl -X POST https://api.linxio.com/api/areas \
  -H "Authorization: Bearer $LINXIO_TOKEN" \
  -H "Content-Type: application/json" \
  --data '{
    "name": "Temporary SDK Test Area",
    "type": "circle",
    "radius": 150,
    "coordinates": [{ "lat": -31.9505, "lng": 115.8605 }]
  }'
```

  </CodeRail>
</ReferenceGrid>
